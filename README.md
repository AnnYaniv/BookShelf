# BookShelf

---

Это финальный проэкт для NIX.

**Bookshelf** - это интернет магазин книг,в нем доступна как бумажная версия книги, так и электронная.


В данном интернет магазине существуют три роли

*Неавторизированный пользователь:*

* Просмотр краткой информации о книге(автор, жанр, издательство, год, количество страниц, isbn, аннотация, цена и т.д).
* Просмотр коментариев от авторизированных пользователей и средней оценки книги.
* Поиск книг(по названию, по жанру, в ценовом диапазоне; доступна сортировка по популярности, по цене).
* Создание аккаунта, авторизация (происходит по паролю и email)

*Авторизированный пользователь:*

* Все тоже что и неавторизированный пользователь +
* Создание заказа
* Оценка книги(можно написать коментарий и оставить оценку к оплаченой книге)
* Список избранного(пользователь будет помещать туда книги которые хочет купить в будущем)

*Админ:*

* Может отменить/принять заказ от авторизированного пользователя
* Добавление книг/редактирование информации о книгах

## Логины и пароли

При запуске приложения происходит миграция данных с помощью flyway. 
Для использования учетной записи админа используйте данные:

    Login: admin@gmail.com
    Password: admin

У всех остальных предустановленных учетных записей пароль: *useruser1*

*P.S. Пароли шифруются благодаря Spring Security*
